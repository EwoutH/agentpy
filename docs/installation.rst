============
Installation
============

For full functionality, agentpy has the following requirements: Python 3.7 or higher, NumPy, scipy, matplotlib, networkx, pandas, SALib, and ipywidgets.

To install the package, run the following command on your console::

	$ pip install agentpy

Alternatively, you can also clone the latest version from GitHub::

	$ git clone hhttps://github.com/JoelForamitti/agentpy.git

The agentpy packet can then be imported as follows (recommended)::

	import agentpy as ap
